# TN 2026 — Voter Intelligence Platform

An independent, non-partisan voter intelligence platform for the **Tamil Nadu 2026 Legislative Assembly Election**.

🌐 **Live Site:** https://karthikkumarm.github.io/TN2026/

---

## What's Inside

| Module | Content |
|---|---|
| **Manifesto Analysis** | 4-party comparison (DMK, ADMK, TVK, NTK), fiscal stress-test, ideal manifesto, issues-vs-promises reality check |
| **Know Your Candidate** | All 4,610 accepted candidates across 233 constituencies, with ECI affidavit links |
| **Deep Dive** | Detailed analysis of 174 promised schemes across 7 thematic sections |

Everything is in a single self-contained HTML file — works 100% offline after first visit, mobile responsive, dark theme, share via WhatsApp.

---

## Data Sources

- **Candidates** → [ECI Affidavit Portal](https://affidavit.eci.gov.in) (self-declared affidavits)
- **Fiscal** → RBI, TN State Finance Budget, CAG
- **Social/Economic** → CGWB · TANGEDCO · DPIIT · IPCC AR6 · NFHS-5

All data is from public-domain sources. This site has no affiliation with any political party or the Election Commission.
